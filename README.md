# 🔥 NEON SURVIVOR
スタイリッシュなネオン世界で、迫りくる敵をかき分けて生き残るトップダウン・サバイバルゲーム。

本作は **1つの HTML ファイル（index.html）だけで動作**する、  
シンプルかつ中毒性の高いサバイバル・ローグライクです。

---

## ✨ 特徴（ここが面白い！）

### ⚔️ 9種類の武器 × 3スロット制
最大3種類の武器を組み合わせて、自分だけのビルドを構築できます。

- **Plasma Gun**（自動射撃）  
- **Scatter Shot**（ショットガン）  
- **Photon Blade**（周囲を回るプラズマ刃）  
- **Void Field**（継続ダメージフィールド）  
- **Railgun**（直線貫通レーザー）  
- **Plasma Grenade**（範囲爆発）  
- **Tesla Coil**（チェーン雷）  
- **Spider Mine**（自走地雷）

---

### 🌀 “Photon Blade” の爽快さがクセになる
高速回転する刃が敵を切り裂き、画面を豪華に彩ります。

---

### 👾 敵の種類と行動パターンがユニーク
Runner / Tank / Triangle / Square / Hexagon など  
視覚で違いが分かる敵が多数登場します。

---

### 🧿 5ウェーブごとに登場する強力ボス

- **Destroyer**  
- **Charger**  
- **Sniper**  
- **Phantom**  
- **Summoner**  
- **Juggernaut**

それぞれ個別の攻撃パターンを持つため、小規模ボスラッシュのような緊張感が楽しめます。

---

### 🔊 Web Audio API を使った “動的BGM”
戦闘状況に合わせてテンポが変化する自動生成サウンドが特徴です。

---
